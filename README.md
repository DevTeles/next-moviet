## Como baixar o projeto


```bash

  # Clonar o repositório
  $ git clone https://github.com/DevTeles/next-moviet.git

  # Entrar no diretório
  $ cd next-moviet

  # Instalar as dependências
  $ yarn

  # Iniciar o projeto
  $ yarn dev
```
